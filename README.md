# DecodeLabs Internship Project 1 - AI Chatbot

## Project Overview

This project is a simple rule-based AI chatbot built using Python as part of the DecodeLabs internship.

The chatbot takes user input and responds with predefined answers based on simple keyword matching and conditional logic. The goal of this project is to practice basic Python programming concepts and understand how a simple chatbot structure works.

## Project Objective

The objective of this project is to create a basic chatbot that can interact with users through text input and provide simple responses.

This project focuses on:

- Using Python functions
- Working with dictionaries
- Handling user input
- Using loops
- Applying conditional statements
- Creating a simple conversation flow

## Files Included

- `Ai_chatbot_p1.py` — Main Python file containing the chatbot code
- `README.md` — Project documentation

## Technologies Used

- Python

## How the Chatbot Works

The chatbot waits for the user to type a message.  
It checks the user input and compares it with predefined keywords or phrases.  
Based on the input, the chatbot returns a suitable response.

The conversation continues until the user chooses to exit.
